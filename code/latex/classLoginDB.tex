\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classLoginDB}{Login\+DB}} class. 

\begin{DoxyAuthor}{Author}
Daniel Herbert
\end{DoxyAuthor}
This class represents the database that will store all user information \begin{DoxyAuthor}{Author}
Daniel Herbert
\end{DoxyAuthor}
This class will connect to an sqlite database and store all user information in the database, as well as the friends a user has added It implements the functions for verifying a user\textquotesingle{}s login credentials, changing a user\textquotesingle{}s password, creating an account, checking if a username already exists, as well as allowing a user to add a friend and subsequently updating that in the database.
